# Startup

---

## Create Python venv

```bash
# On Windows
python -m venv .venv
.\.venv\Scripts\activate

# On MacOS and Linux
python -m venv .venv
source .venv/bit/activate
```

## Insgtall Libaries

```bash
pip install -r requirements.txt
```

## Select Kernel

```bash
# On Windows
.\.venv\Scripts\activate

# On Mac OS and Linux
.venv/bit/python
```
